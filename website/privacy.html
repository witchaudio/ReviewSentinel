Privacy Policy

Last updated: March 2026

ReviewSentinel is a Chrome extension that detects potentially fake reviews on Amazon. This policy explains what data we handle and how.

What we collect

When you visit an Amazon product or review page, the extension reads the following fields from each visible review:

  • Review text body
  • Star rating (1–5)
  • Whether the purchase is verified
  • Number of helpful votes
  • Length of the review text

This data is sent to our scoring API at reviewsentinel-api.onrender.com to generate a trust score and explanation. No other data is collected or transmitted.

What we do NOT collect

  • Your name, email, or any account information
  • Your browsing history or any pages other than Amazon review pages
  • Any personally identifiable information
  • Payment or purchase data

Data retention

Review text sent to our API is processed in memory to produce a score and is not stored, logged, or retained in any database. Each request is stateless and leaves no record.

Third-party services

The API is hosted on Render. Render may retain standard server access logs (IP address, timestamp, request path) for a limited period per their own privacy policy. We do not use any analytics, advertising, or tracking services.

chrome.storage

The extension stores your configured API endpoint URL in chrome.storage.sync. This is synced across your Chrome devices by Google. No review content is stored locally or in sync storage.

Open source

The full source code for both the extension and the API is publicly available at github.com/witchaudio/ReviewSentinel. You can audit exactly what data is sent and how scores are computed.

Changes

If we update this policy, we will update the "Last updated" date above and, for material changes, note it in the GitHub release notes.
